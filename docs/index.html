Getting Start

Pour commencer, il faut configurer votre base de données grâce à un fichier json. Par défaut celui-ci se trouve Dans le dossier Attila/conf/Db.conf

Sur Venus 3, vous trouverez celui-ci dans /bundles/conf/Db.conf

Voici à quoi ressemble le fichier de configuration d'une base de données :

{
    "configuration": {
        "type": "mysql",
        "host": "127.0.0.1",
        "db": "attila",
        "user": "root",
        "password": "root",
        "tables": {
            "user": {
                "fields": {
                    "id": {
                        "type": "int",
                        "key": "primary",
                        "null": false,
                        "undefined": true,
                        "autoincrement": true
                    }
                }
            }
        }
    }
}
La connexion se configure de façon classique comme vous pouvez le constater ci-dessus. Il est essentiel ensuite de configurer les tables de notre projet (nosu verrons cela en détail plus tard.

Attila vous propose différents outils pour automatiser la créaetion des tables, des entités et des modèles (équivalent à un repository).
